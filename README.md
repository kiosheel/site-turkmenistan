# Projet Documentaire : Turkménistan


Ce site web a été conçu dans un double objectif :
1. **Apprentissage technique** : Pratiquer les bases du développement web (HTML5/CSS3) et comprendre le processus de déploiement via Vercel.
2. **Sensibilisation** : Mettre en lumière le Turkménistan, un pays souvent méconnu où règnent la dictature, la censure et un climat de peur pour la population.

- **Structure Sémantique** : Utilisation des balises HTML5 (`<header>`, `<main>`, `<section>`) pour un code propre et accessible.
- **Design & Layout** : Gestion du responsive design et intégration de polices via Google Fonts.
- **Navigation Multi-pages** : Organisation de liens fluides entre les différentes sections du site.
- **Déploiement Continu** : Mise en ligne automatisée avec Vercel.

Le site explore plusieurs facettes du pays :
- La culture et les traditions culinaires.
- L'architecture monumentale d'Achgabat.
- La réalité sociale et les violations des droits de l'homme (sourcé par RSF et Human Rights Watch).
